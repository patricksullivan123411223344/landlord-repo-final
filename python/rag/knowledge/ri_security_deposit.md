# Rhode Island Security Deposit Notes

Rhode Island security deposit handling is commonly discussed under RIGL landlord-tenant rules. A frequent tenant issue is delayed return of deposit funds and missing itemized deductions.

Practical documentation steps:
- Keep move-in and move-out photos with timestamps.
- Request itemized deductions in writing.
- Keep a copy of your lease and all notice emails/texts.
- Use certified mail for formal demand letters when necessary.

Potential risk indicators:
- No written itemized deductions
- Broad charges for normal wear and tear
- No response to written requests
